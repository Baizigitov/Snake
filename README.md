# Название проекта 
Змейка

**Описание:**  
Простая игра Змейка

**Технологии:**  
- Python 
- Pygame
 

**Установка:**  
1. Клонируйте репозиторий: `git clone https://github.com/ваш-ник/проект`  
2. Установите зависимости: `pip install -r requirements.txt`  
3. Запустите: `python main.py`
